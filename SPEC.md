# Nobulex Protocol Specification

The full Nobulex Covenant Protocol specification lives in [SPECIFICATION.md](./SPECIFICATION.md).

It covers:

- Covenant document schema and JSON structure
- Covenant Constraint Language (CCL) grammar and evaluation semantics
- Canonical form algorithm (RFC 8785 JCS)
- Document ID computation (SHA-256 content addressing)
- Ed25519 signing and verification
- 11-check verification algorithm
- Chain composition with monotonic constraint narrowing
- Countersignature protocol
- Protocol constants and cryptographic requirements
- Complete test vectors for cross-implementation validation
- JSON Schema definitions (Draft 2020-12)
